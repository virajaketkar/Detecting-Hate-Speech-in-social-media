Goal

The goal of this project is to understand the components of a Machine Learning Life Cycle in order to apply them to practical business problem.

We will tackle unstructured data and cover advanced algorithms for complex problems that require specialised methods. We will apply advanced algorithms such as ensemble methods, deep neural networks, and sequential algorithms using Python

  • Plan each step of the machine learning life-cycle by identifying business problem, evaluating constraints and resources, deciding on appropriate approaches, selecting several potential machine learning models, building a minimal viable product prototype, and determining the outcome for success.
  • Explore data (structured and unstructured) by performing basic exploration such as visualise and analyse the data, computing exploratory statistics, extracting and design features (carry out feature selection/extraction/engineering), and reducing the dimensions.
  • Select the appropriate machine learning models to match to the type of problem, know the trade-off (shortfalls and strengths) of each, compute proper evaluation metrics and explain and visualise results, iterate on prototypes strategically to improve performance (i.e. baseline to ensemble).

Evaluation Metrics

We report the performance of each tweet analysis tool in terms of precision, recall, and f-measure for all classes.
